# Project: ProfitVision AI

## Overview
ProfitVision AI is a multimedia web project developed for the ICT course ("Информационно-коммуникационные технологии") at Narxoz University. The project aims to demonstrate the application of AI tools and web technologies in creating a functional, interactive, and visually appealing landing page focused on business analytics (specifically Customer Retention/Churn Rate Analysis).

## Main Goal
To develop a 4-5 page website that integrates mandatory AI components and provides analytical insights into client retention strategies, adhering to the university's academic requirements for the 2025-2026 academic year.

## Core Technologies
- **Framework:** React 18 (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Minimalist style)
- **UI Components:** shadcn/ui (Radix UI)
- **State Management:** TanStack Query (React Query)
- **Routing:** React Router DOM
- **AI Integration:** Google Generative AI (@google/generative-ai)

## Mandatory AI & Interactive Components
1. **Hero Section:** Highlighting the project's core value proposition.
2. **AI-Generated Content:** Images/illustrations and analytical text blocks.
3. **AI Agent:** Interactive component for data-driven insights.
4. **Dynamic Data:** "What-If" tables and charts for scenario planning.
5. **Multimedia:** Video (AI-avatar/animation) and background AI music with controls.
6. **Interaction:** A functional Chatbot widget.

## Conventions & Standards
- **Design:** Follow a minimalist aesthetic with harmonic color schemes.
- **Components:** Functional components with TypeScript interfaces. Use shadcn/ui primitives for consistency.
- **Accessibility:** Ensure buttons/links meet minimum touch targets and have ARIA labels.
- **Structure:**
  - `src/components/features`: High-level feature blocks.
  - `src/components/layout`: Global layout components (Header, Footer).
  - `src/components/ui`: Base UI primitives.
  - `src/pages`: Main application routes.

## Deployment & Verification
- The project is designed for demo-mode presentation.
- All links and buttons must be functional.
- The Footer must contain: Author Name, Discipline (ИКТ), University (Narxoz), Program, and Year.

## Development Workflow
1. **Research:** Align implementation with "Требования и темы проектов" PDF.
2. **Implementation:** Focus on fulfilling all 8 mandatory AI component requirements.
3. **Validation:** Ensure the site consists of at least 4-5 logically connected pages.
